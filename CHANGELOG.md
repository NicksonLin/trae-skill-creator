# Changelog

## v1.0.0

Initial public release of `trae-skill-creator`.

### Added

- Skill creation workflow based on single responsibility and clear trigger boundaries
- Metadata rules for skill name and description
- Five core standards for reviewing skill quality
- Progressive disclosure guidance for large skills
- Evaluation-driven workflow for failure-first iteration
- Anti-pattern checklist for common skill design mistakes
- Examples library with good and bad skill design patterns
- Third-party attribution for Trae and Anthropic source material

### Intended Users

- Trae users building custom skills
- AI agent workflow designers
- Prompt engineers designing reusable agent instructions
- Teams standardizing internal skill authoring practices
